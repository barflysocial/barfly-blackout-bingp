# Battle Bingo v2 — New Rules Build

This build includes the TV-fit fixes plus the latest gameplay updates:

- Calls use WIN format: W1-W75, I1-I75, and N1-N75.
- A call only marks the matching W/I/N column and number.
- The call pool has 225 possible calls, but the game still ends when a player reaches the win condition first.
- Fire is passive/automatic: if a player has Fire and Freeze is used against them, Fire blocks the Freeze and is consumed.
- Recover is passive/automatic: after a call ends, if a player missed a called number on their card and has Recover, Recover marks it and is consumed.
- Freeze remains the active attack power and targets the ranked opponent according to the existing rank rule.
- Players earn one power every 3 cleared rows, can hold up to 3 powers, and cycle Freeze → Fire → Recover before repeating.
- The public Share button now opens a panel with a QR code, Copy Link, and native Share. The QR code uses the same URL as the share/play link.
- New sessions default to the WIN call board.
