How to Play

  1. Everyone gets exactly 3 random 5x5 BINGO cards. There are no free spaces.
  2. The host chooses the mode: Blackout = 1 card, Double = 2 cards, Triple = all 3 cards.
  3. Numbers are called every 7 seconds.
  4. Orange means the number was called and is available. Green means you selected it.
  5. You may select only 1 number during each 7-second call window.
  6. Powers unlock on a card after its 3rd valid BINGO. Only horizontal or vertical BINGOs count. No diagonals.
  7. The first BINGO on a card locks it as a Horizontal or Vertical power card.

Powers

Freeze freezes the current called number for everyone else. You can still select it.

Shield blocks a Freeze so you can still select the frozen number.

Swap moves a green selected number from one of your cards to the same orange number on another card.

Steal takes one random unused power from the leaderboard leader, or the next ranked player with a power.
