How to Play

Read this before the host starts the round.

Lobby
  1. Everyone gets exactly 3 random 5x5 BINGO cards each round.
  2. The host chooses the win mode: Blackout = 1 card, Double Blackout = 2 cards, Triple Blackout = all 3 cards.
  3. When a number is called, matching squares turn orange. Orange means called, but not selected.
  4. You may select only one current called number during each 7-second call window. Your selected square turns green.
  5. Powers are earned under each card. The first BINGO locks that card as Horizontal or Vertical. No diagonals count for powers.
  6. Powers do not begin until that card reaches its 3rd valid BINGO. The 3rd, 4th, and 5th valid BINGOs each award one random power.
❄ FreezeFreeze the current called number for everyone else. You can still select it.
🛡 ShieldBlocks one Freeze so you can still select a frozen call.
↔ SwapMove one green mark to the same orange called number on another card.
🎭 StealSteal one random unused power from the highest-ranked player who has one.
