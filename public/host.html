Directions Host Should Read Before Start

  1. Players scan the QR code and join with name + session code.
  2. Everyone gets exactly 3 random 5x5 BINGO cards. No free spaces.
  3. The host mode controls the win: Blackout = 1 card, Double = 2 cards, Triple = all 3 cards.
  4. Orange means called/highlighted. Green means selected/official.
  5. Players may select only one current number during each 7-second call window. No auto-select.
  6. Powers are earned per card only after the 3rd valid BINGO. First BINGO locks that card to horizontal or vertical. No diagonals count for powers.
  7. Freeze freezes the current call for everyone else. Shield blocks Freeze. Swap moves a green mark to the same orange number on another card. Steal steals a random unused power from the leaderboard leader or next ranked player with a power.
