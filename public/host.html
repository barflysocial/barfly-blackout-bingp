Directions Host Should Read Before Start

  1. Players scan the QR code and join with name + session code.
  2. Everyone gets exactly 3 random 5x5 BINGO cards.
  3. The host mode controls the win: Blackout = 1 card, Double = 2 cards, Triple = all 3 cards.
  4. Orange means called/highlighted. Green means selected/official.
  5. Players may select only one current number during each 7-second call window.
  6. Powers are earned per card only after the 3rd valid BINGO. First BINGO locks that card to horizontal or vertical. No diagonals count for powers.
  7. Random powers are Freeze, Shield, Swap, and Steal.
❄ FreezeFreezes the current called number for everyone else. The player who used it can still select the number.
🛡 ShieldAutomatically blocks one Freeze so the player can still select the frozen call.
↔ SwapMoves one green mark to the same orange called number on another card.
🎭 StealSteals one random unused power from the highest-ranked player who has a power.
